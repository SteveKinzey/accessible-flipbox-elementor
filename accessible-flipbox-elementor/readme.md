# Accessible Flipbox for Elementor

This plugin adds keyboard accessibility to Elementor Flipbox widgets:
- Adds ARIA attributes
- Enables toggling via Enter/Space keys
- Improves screen reader support

## Installation
1. Download the ZIP.
2. In WP Admin → Plugins → Add New → Upload.
3. Activate.

## Testing
1. Tab through page — Flipbox is focusable.
2. Press Enter or Space — it flips.
3. Screen reader announces expanded/collapsed state.
